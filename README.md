# Paradise Masonry - Modern Website Copy

This is a modern reconstruction of the Paradise Masonry website, built with **Next.js**, **React**, **Tailwind CSS**, and **TypeScript**.

## Features
- **Modern UI**: Clean, professional design focused on commercial concrete services.
- **Responsive**: Fully functional on mobile, tablet, and desktop.
- **Next.js 16 (Turbopack)**: Fast development and optimized production builds.
- **Tailwind CSS v4**: Modern styling with optimized CSS.

## Tech Stack
- Framework: [Next.js](https://nextjs.org/)
- Styling: [Tailwind CSS](https://tailwindcss.com/)
- Icons: [Lucide React](https://lucide.dev/)
- Components: React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) (or the port specified in the terminal) to see the result.

## Project Structure
- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components (Navbar, Hero, Services, etc.).
- `public`: Static assets.
